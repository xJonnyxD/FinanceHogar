## 📞 ¿QUÉ HACER SI ALGO FALLA?

### "La demo no funciona"
→ Di: "A veces hay latencia. Déjame mostrarles el código en Swagger. Aquí pueden ver la respuesta JSON que genera la alerta..."

### "Olvido qué dije"
→ Pausa 3 segundos. Respira. Mira al evaluador. Pregunta: "¿Puedo reformular eso?" Y continúa.

### "El tribunal pregunta algo que no esperaba"
→ Respuesta honesta: "Excelente pregunta, no lo tengo documentado, pero mi pensamiento es [tu respuesta lógica]."

### "Falla el proyector"
→ Di: "Continuamos sin proyector. Yo explico el flujo mientras ustedes siguen en laptop." (Tenes script + FAQ impreso)
